# Mis gastos backend

Node.js backend for Mis gastos project. It's based on [Node JS Server](https://github.com/brunopk/nodejs-server) to run as a Home Assistant add-on.

## Development

### Development with HMR (hot module reloading)

```bash
yarn dev
```
