# Scripts

Mis Gastos Backend includes the following Python scripts :

- [`/scripts/import_spends_from_csv.py`](/scripts/import_spends_from_csv.py): Facilitates data migration from a similar application built with Google Apps Script.

> The environment variables `MIS_GASTOS_ADMIN_JWT_CLIENT_ID` and `MIS_GASTOS_ADMIN_JWT_CLIENT_SECRET` are for the client ID and client secret of the Client Credentials flow. For more information about Client Credentials flow refer to [`/doc/security.md`](/doc/security.md).
